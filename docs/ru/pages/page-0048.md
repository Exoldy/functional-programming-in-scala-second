## 19 2.1 Знакомство с языком Scala

Метод `printAbs` помечен аннотацией `@main` — это как неоновая вывеска для компилятора: «Эй, Scala, вот точка входа в нашу эпопею!». Аргументов не жрёт, возвращает `Unit`. Scala, конечно, парсит аргументы командной строки, но мы тут не мудствуем. `Unit` играет ту же роль, что и `void` в C или Java — стартовая площадка для всей хуйни. В Scala каждый метод обязан что-то выплюнуть, если не крашнется или не зависнет в вечном цикле, как твой первый рекурсивный факториал без хвостовой оптимизации. А `printAbs` ничего осмысленного не возвращает, вот и придумали специальный тип `Unit` — для таких процедурных костылей. У него ровно одно значение, литерал `()`, пара пустых скобок (читается *unit*, как и тип сам). Обычно `Unit` в сигнатуре — это намёк: «Тут сайд-эффекты, братан, держи консоль или I/O». Тело нашего `printAbs` печатает на консоль строку `String`, которую возвращает вызов `formatAbs`. Кстати, `println` возвращает `Unit` — идеально ложится в то, что ждёт от нас `printAbs`, как пазл из девяностых.

### 2.1.1 Запуск нашей программы

Тут разберём самый примитивный способ запустить Scala-прогу — для коротких примеров, чтоб не ебаться с настройками, как в первый раз с sbt. В реальной жизни строишь и запускаешь через `scala-cli` — это REPL-командочку (REPL — Read-Eval-Print Loop) для общения со Scala, или `sbt`, король билд-тулов для нас. Или вообще IDE: IntelliJ, VS Code с плагином — и живи. Есть и веб-варианты, типа Scastie (http://scastie.scala-lang.org). Подробности по установке `scala-cli` и `sbt` — в репозитории книги на GitHub (https://github.com/fpinscala/fpinscala). Самый простой способ — с терминала, напрямую дергаем компилятор Scala. Сначала скачай Scala 3 с офсайта по ссылке https://docs.scala-lang.org/getting-started... Установил? Кидай код в файл `MyProgram.scala` или как там назовёшь. Компилируем в JVM-байткод командой `scalac`:

```scala
> scalac MyProgram.scala
```

Это навалит файлы с суффиксом `.class` и `.tasty`<sup>3</sup>. Эти байты жрёт JVM. Запускаем через `scala`, указав имя класса с `@main`:

```scala
> scala printAbs
The absolute value of -42 is 42.
```

На самом деле, компилировать заранее `scalac` не обязательно. Для такой простой хуйни хватит интерпретатора — кидай прямо в `scala`:

![Страница 0048, изображение 1](./assets/page-0048/image-01.png)

```scala
> scala MyProgram.scala
The absolute value of -42 is 42.
```

<sup>3</sup> Подробнее про файлы `.tasty` — тут: https://mng.bz/vXX1.
