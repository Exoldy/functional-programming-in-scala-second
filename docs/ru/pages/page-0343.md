### 12.1 Обобщение монад

К этому моменту мы уже заебались видеть, как всякие sequence и traverse впаривают по сто раз для разных монад, а в прошлом чаптере мы их наконец обобщили — чтоб работали для любой монады F, как нормальные пацаны:

```scala
def sequence[A](fas: List[F[A]]): F[List[A]]
traverse(fas)(fa => fa)
def traverse[A, B](as: List[A])(f: A => F[B]): F[List[B]]
as.foldRight(unit(List[B]()))((a, acc) => f(a).map2(acc)(_ :: _))
```

Здесь `traverse` лепится из `map2` и `unit`, и мы видели, что `map2` — это просто flatMap в другой обёртке, как конструктор из лего:

```scala
extension [A](fa: F[A])
def map2[B, C](fb: F[B])(f: (A, B) => C): F[C] =
fa.flatMap(a => fb.map(b => f(a, b)))
```

Может, вы не врубились сразу, но половина полезных комбинаторов на `Monad` набивается чисто из `unit` и `map2`, без этой монадической хуйни вроде flatMap. `Traverse` — яркий пример: он flatMap напрямую не ебёт, ему похуй, примитив map2 или дерив. Плюс, для кучи дата-типов `map2` лепишь напрямую, без flatMap-цирка. Всё это орёт: "давайте мутируем Monad!" Там flatMap и unit — примитивы, map2 деривится, а мы перевернём: unit и map2 как основа. Получится новая абстракция — *аппликативный функтор*, послабее монады, как lite-версия, но с ограничениями приходят плюшки, пацаны, поверьте опыту 16 лет в проде.

### 12.2 Трейт `Applicative` (аппликативный функтор)

Аппликативные функторы ловим в свежий трейт `Applicative`, где `unit` и `map2` — чистые примитивы, без лишнего жира.

**Листинг 12.1.** Создаём интерфейс `Applicative`

```scala
trait Applicative[F[_]] extends Functor[F]:
// primitive combinators
def unit[A](a: => A): F[A]
extension [A](fa: F[A])
def map2[B, C](fb: F[B])(f: (A, B) => C): F[C]
```

![Страница 0343, изображение 1](./assets/page-0343/image-01.png)

> Можем `map` набить чисто из `unit` и `map2`, как по шаблону.

```scala
// derived combinators
extension [A](fa: F[A])
def map[B](f: A => B): F[B] =
fa.map2(unit(()))((a, _) => f(a))
```

> Определение `traverse` — один хуй, идентичное.  
> Вспомните, `()` — единственный житель типа `Unit`, так что `unit(())` — это вызов unit с дамп-значением `()`.

```scala
def traverse[A, B](
as: List[A])(f: A => F[B]): F[List[B]] =
as.foldRight(unit(List[B]()))((a, acc) => f(a).map2(acc)(_ :: _))
```
