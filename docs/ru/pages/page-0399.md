### 13.4.1 Свободные монады

Конструкторы ``Return`` и ``FlatMap`` — это прямые свидетели, что тип данных монадный для *любого* выбора ``F`. И раз это ровно те приёмы, что генерят монаду без лишней хуйни, мы и зовём её *свободной* монадой.<sup>10</sup> Блядь, как байк без рамы — чистая свобода, без оков собственной структуры.

![Страница 0399, изображение 1](./assets/page-0399/image-01.png)

#### УПРАЖНЕНИЕ 13.1

``Free`` — монада для любого выбора ``F``. Реализуй методы ``map`` и ``flatMap`` на енуме ``Free`` и выдавай инстанс ``Monad`` для ``Free[F,_]``. Пацаны, это фундамент, как `flatMap` в `List`, без него вся постройка рухнет:

```scala
given freeMonad[F[_]]: Monad[[x] =>> Free[F, x]] with
def unit[A](a: => A): Free[F, A] = ???
extension [A](fa: Free[F, A])
def flatMap[B](f: A => Free[F, B]): Free[F, B] = ???
```

![Страница 0399, изображение 2](./assets/page-0399/image-02.png)

#### УПРАЖНЕНИЕ 13.2

Сделай спец хвостово-реккурсивный интерпретатор ``runTrampoline``, чтоб гонять ``Free[Function0,`` ``A]`` без стек-оверфлоу — чисто, как в продакшене, где JVM не прощает:

![Страница 0399, изображение 3](./assets/page-0399/image-03.png)

```scala
extension [A](fa: Free[Function0, A])
@annotation.tailrec
def runTrampoline: A = ???
```

#### УПРАЖНЕНИЕ 13.3

*Хардкор*: Реализуй универсальный интерпретатор для ``Free[F,`` ``A]`` по данному ``Monad[F]``. Определи его как метод на типе ``Free``. Ориентируйся на интерпретатор ``Async`` выше, включая хвостово-реккурсивный ``step`` — чтоб стек не рвал жопу на больших цепочках:

```scala
enum Free[F[_], A]:
...
def run(using F: Monad[F]): F[A] = ???
@annotation.tailrec
final def step: Free[F, A] = ???
```

<sup>10</sup> Тут *free* — это «свободно генерируемая», в смысле ``F`` не обязан нести монадическую ношу сам по себе, никаких цепей. За строгим определением лезь в нотсы главы (https://github.com/fpinscala/fpinscala/wiki), там всё по матану разжёвано.
