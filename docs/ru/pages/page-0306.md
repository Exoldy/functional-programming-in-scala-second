## 277 10.9 Ответы на упражнения

```scala
def combine(wc1: WC, wc2: WC) = (wc1, wc2) match
case (WC.Stub(a), WC.Stub(b)) => ???
case (WC.Stub(a), WC.Part(l, w, r)) => ???
case (WC.Part(l, w, r), WC.Stub(a)) => ???
case (WC.Part(l, w, r), WC.Part(l2, w2, r2)) => ???
```

Склеиваем две заглушки — и бац, новая конкатенированная заглушка, потому что целого слова мы ещё не просекли, как слепые котята в тумане:

```scala
case (WC.Stub(a), WC.Stub(b)) => WC.Stub(a + b)
```

Заглушку с частью или часть с заглушкой лепим в новую часть, растягивая либо `lStub`, либо `rStub` на эту заглушку. Главное — не перепутай порядок символов, чтоб не вышло как в том меме с left-pad — например, заглушка слева от части растягивает левую заглушку в результате, чтоб всё по делу шло:

```scala
case (WC.Stub(a), WC.Part(l, w, r)) => WC.Part(a + l, w, r)
case (WC.Part(l, w, r), WC.Stub(a)) => WC.Part(l, w, r + a)
```

Наконец, две части: берём левую заглушку от левой как левую новую, правую от правой — как правую, суммируем счётчики слов. А *внутренние заглушки* — правую от левой части и левую от правой — склеиваем и, если не пустышка, считаем за слово, чтоб не упустить эту хитрую хуйню посередине:

```scala
case (WC.Part(l, w, r), WC.Part(l2, w2, r2)) =>
WC.Part(l, w + (if (r + l2).isEmpty then 0 else 1) + w2, r2)
```

Собираем всю эту шарманку в один `combine`, и вот она, красота:

```scala
val wcMonoid: Monoid[WC] = new Monoid[WC]:
val empty = WC.Stub("")
def combine(wc1: WC, wc2: WC) = (wc1, wc2) match
case (WC.Stub(a), WC.Stub(b)) => WC.Stub(a + b)
case (WC.Stub(a), WC.Part(l, w, r)) => WC.Part(a + l, w, r)
case (WC.Part(l, w, r), WC.Stub(a)) => WC.Part(l, w, r + a)
case (WC.Part(l, w, r), WC.Part(l2, w2, r2)) =>
WC.Part(l, w + (if (r + l2).isEmpty then 0 else 1) + w2, r2)
```

Откуда знать, что мы не накосячили, как в том классическом код-ревью, где всё казалось ок, а на тесте сломалось? Проверяем законами монойда из упражнения 10.4, по-настоящему, по-пацански. Для этого слепим `Gen[WC]`:

```scala
def wcGen: Gen[WC] =
val smallString = Gen.choose(0, 10).flatMap(Gen.stringN)
val genStub = smallString.map(s => WC.Stub(s))
val genPart = for
lStub <- smallString
words <- Gen.choose(0, 10)
rStub <- smallString
yield WC.Part(lStub, words, rStub)
Gen.union(genStub, genPart)
```
