## 437 15.3 Расширяемые пулы и стримы

```scala
object Ref:
  def apply[F[_], A](initial: A)(using F: Monad[F]): Ref[F, A] =
    new Ref(
      new AtomicReference[A](initial),
      [A] => (th: () => A) => F.unit(()).map(_ => th())
    )
```

`Ref[F, A]` — это чисто функциональный аналог переменной типа `A`, где весь доступ и мутации упакованы в эффект `F`, чтоб не размазывать состояние по всему коду, как сперму по монитору после дедлайна. Реализация юзает под капотом `java.util.concurrent.AtomicReference` для хранения, а наружу выставляет эффектовые геттеры и сеттеры — чисто, атомарно, без состояний гонки (race conditions), которые всех нас когда-то ебали в проде. Операция `modify` — это вообще огонь: атомарно берёт текущее значение, вычисляет новое и позволяет прицепить любое дополнительное дерьмо в ответ. С помощью `Ref` и `modify` мы лепим `Scope` как машину состояний с фазами `Open` и `Closed` — классика, пацаны, я через такое в 2010-м на Cats Effect 0.x ковырялся, пока не понял, как оно спасает от утечек:

```scala
class Scope[F[_]](
  parent: Option[Scope[F]],
  val id: Id,
  state: Ref[F, Scope.State[F]]
)(using F: MonadThrow[F]):
  import Scope.State
  def open(finalizer: F[Unit]): F[Scope[F]] = ???
  def close: F[Unit] = ???
object Scope:
  enum State[F[_]]:
    case Open(
      finalizer: F[Unit],
      subscopes: Vector[Scope[F]]
    ) extends State[F]
    case Closed() extends State[F]
  def root[F[_]](using F: MonadThrow[F]): Scope[F[_]] =
    new Scope(
      None,
      new Id,
      Ref(State.Open(F.unit(()), Vector.empty))
    )
```

`Scope` пихает своё mutable состояние в `Ref[F, State[F]]` — чтоб не потерять, как ключи от тачки после корпора. Тип `State[F]` — это enum из двух случаев: `Open` и `Closed`. `Open` держит реф на финализатор для этого скоупа и всех подскоупов — типа, "я закрою все дыры, даже если ты забудешь". Подскоуп открываем, modifyнув новое состояние `Open` с свежим реестром — и вуаля, ресурсы в безопасности, без тех "ой, блядь, GC не справился" из imperative ада:

```scala
def open(finalizer: F[Unit]): F[Scope[F]] =
  state
    .modify:
      case State.Open(myFinalizer, subscopes) =>
        val sub = new Scope(Some(this), new Id,
                            Ref(State.Open(finalizer, Vector.empty)))
        State.Open(myFinalizer, subscopes :+ sub) -> F.unit(sub)
      case State.Closed() =>
        val next = parent match
          case None =>
            F.raiseError(new RuntimeException("root scope already closed"))
```
