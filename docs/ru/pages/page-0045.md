Эта глава в основном для тех пацанов, кто только-только влетает в Scala, FP или вообще в этот цирк с конями. Погружение — как удар током для изучения иностранного, так что ныряем с головой, без спасательных жилетов. Единственный способ, чтоб Scala-код не выглядел как иероглифы из параллельной вселенной, — это наглядеть его до посинения. В первой главе мы уже покурили примерами, а здесь стартуем с маленькой, но полной проги — как первый блин, но без комков. Потом разберём по винтикам, чтоб разжевать, что она делает, и въехать в базовые фичи языка Scala и его синтаксис. Цель книги — вкатить вас в FP по полной, Scala — наш ржавый понтиак для этой дороги, так что базовый арсенал синтаксиса подкачать надо. Освоили основы — прёмся к техникам функционального кодинга: циклы через *хвостово-рекурсивные функции (tail-recursive functions)*, *функции высшего порядка (higher-order functions)*. Высшие — это такие уроды, что жрут другие функции как аргументы и могут пернуть функцией в ответ. Покажем примеры *полиморфных (polymorphic)* высших, где типы ведут за ручку прямиком к имплементации, как GPS в тумане. Много мяса в этой главе, высшие функции могут мозг взорвать, если вы из мира, где функции не таскают как чемоданы. Не парьтесь, не обязательно глотать всё подряд или решать каждую задачку — мы вернёмся к этому дерьму с других углов по всей книге, а здесь просто первая доза для иммунитета.

### 2.1 Знакомство с языком Scala

Вот полная Scala-прога, которую разберём по косточкам, как на код-ревью после трёх банок энергетика. Новых FP-фишек не вводим — чисто язык и синтаксис, чтоб не путаться в ногах.

![Страница 0045, изображение 1](./assets/page-0045/image-01.png)

**Листинг 2.1** Простая Scala-программа

> Объявляет *singleton-объект (singleton object)* — это как класс и его единственный экземпляр в одном флаконе, чистый объект-ориентальный хак.

```scala
// A comment!
/* Another comment */
/** A documentation comment */
object MyProgram:
def abs(n: Int): Int =
if n < 0 then -n
else n
```

> `abs` жрёт `Int` и сплёвывает `Int`.

> Если `n` меньше нуля — возвращает negation, то бишь `-n`.

> Приватный метод — только свои в `MyProgram` могут его тронуть. Тип возврата опционален, здесь выводится автоматически (inferred) из последней строчки в блоке, лениво и красиво.

```scala
private def formatAbs(x: Int) =
val msg = "The absolute value of %d is %d"
msg.format(x, abs(x))
```

> Строка с двумя плейсхолдерами `%d` для чисел, как в старом добром `printf`.

> Заменяет эти `%d` на `x` и `abs(x)` по очереди, классика форматирования.

```scala
@main def printAbs: Unit =
println(formatAbs(-42))
```

> `Unit` — это как `void` в Java или C, пустышка для "ничего не вернул".
