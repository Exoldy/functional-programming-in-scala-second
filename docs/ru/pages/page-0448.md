## 419 15.2 Простые трансформации потоков

Как мы уже хренову тучу раз видели в этой книге, когда вылавливаешь общие паттерны в пачке функций, выноси их в generic-комбинаторы, чтоб не дрочить копипасту каждый раз заново. Функции ``count``, ``tally`` и ``slidingMean`` — все на одной волне: у каждой по куску состояния, функция перехода, которая апдейтит это состояние в ответ на инпут, и выдает один-единственный аутпут. Обобщаем эту хуйню в комбинатор, назовем его ``mapAccumulate`` — классика, блядь, как `fold` в любой императивной помойке, только здесь без побочек:

```scala
def mapAccumulate[S, O2](init: S)(f: (S, O) => (S, O2)): Pull[O2, (S, R)] =
  uncons.flatMap:
    case Left(r) => Result((init, r))
    case Right((hd, tl)) =>
      val (s, out) = f(init, hd)
      Output(out) >> tl.mapAccumulate(s)(f)
```

Перепиши ``count``, ``tally`` и ``slidingMean``, используя ``mapAccumulate`` вместо явной рекурсии. Ха, пост-ирония: `fold` через `fold`, чтоб почувствовать вкус дзена.

### Монада Pull

Операция ``flatMap`` вместе с конструктором ``Result`` формируют инстанс монады для ``[x] =>> Pull[O, x]``:

```scala
given [O]: Monad[[x] =>> Pull[O, x]] with
  def unit[A](a: => A): Pull[O, A] = Result(a)

extension [A](pa: Pull[O, A])
  def flatMap[B](f: A => Pull[O, B]): Pull[O, B] =
    pa.flatMap(f)
```

Можем слепить альтернативный инстанс монады, взяв ``Output`` как реализацию ``unit`` (единица) и придумав другую для ``flatMap``. ``Output(a)`` дает нам ``Pull[A, Unit]`` — резалтинг тип ``Unit``, так что давай определим ``Monad[[x] =>> Pull[x, Unit]]``. Подставляем этот тип в определение ``Monad``, следуем по типам — и бац, нужна версия ``flatMap`` с такой сигнатурой:

```scala
extension [O](self: Pull[O, Unit])
  def flatMap[O2](f: O => Pull[O2, Unit]): Pull[O2, Unit]
```

Заметил, функция, которую пихаем в ``flatMap``, жрет аутпут из оригинального ``Pull``. Это намекает, что наша реализация ``flatMap`` будет как те монады для ``List`` и ``LazyList`` — вызываем функцию на каждом элементе аутпута и конкатим все резалты в один жирный поток.
