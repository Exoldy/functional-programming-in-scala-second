> Часть 2: Функциональный дизайн и библиотеки комбинаторов / Глава 9: Парсерные комбинаторы / 9.8 Ответы на упражнения

Одна хуйня с этой версией `array` — мы пробелы не жрём, как ни в чём не бывало. 
Плюс, наши токен-парсеры (вроде `[`, `]` и `,`) — committed (committed parsers) на всю катушку: 
если не матчатся, то весь парсинг летит к чертям собачьим, без шанса на альтернативу через `or`. 
Короче, давай новый конструктор замутим, чтоб и пробелы сожрал, и committed-хуйню пофиксил — 
как нормальный FP-комбинатор, который не подводит в проде:

```scala
def token(s: String): Parser[String] =
  string(s).attempt <* whitespace

def array: Parser[JSON] =
  token("[") *> value.sep(token(",")).map(vs =>
    JArray(vs.toIndexedSeq)
  ) <* token("]")
```

Наконец, заюзаем `scope`, чтоб когда массив не спарсится, выдал не просто "fail", 
а жирную ошибку с деталями — чтоб дебаггить было как по маслу, а не как в тёмном лесу:

```scala
def array: Parser[JSON] = (
  token("[") *>
  value.sep(token(",")).map(vs =>
    JArray(vs.toIndexedSeq)
  ) <*
  token("]")
).scope("array")
```

Теперь `value` реализуем. JSON-значение — это либо литерал, массив или объект, классика жанра. 
Литерал — одно из:

* `null`
* число
* строка в кавычках
* `true` или `false`.

Похер, лепим парсер на прямую, без лишней бюрократии:

```scala
def value: Parser[JSON] = lit | obj | array
def lit: Parser[JSON] = (
  token("null").as(JNull) |
  double.map(JNumber(_)) |
  escapedQuoted.map(JString(_)) |
  token("true").as(JBool(true)) |
  token("false").as(JBool(false))
).scope("literal")
def double: Parser[Double] = ???
def escapedQuoted: Parser[String] = ???
```

`double` и `escapedQuoted` тут недописаны (я сам через такое проходил — лень допиливать в книге), 
но полные варианты в ответах на GitHub-репозитории, чекни.

Наконец, `obj` из начала главы допилим. JSON-объект — это запятая-разделённый список пар ключ-значение 
в фигурных скобках `{ }`, где ключ всегда строка в кавычках, а значение от него отделено токеном `:` — 
как нормальный JSON, без самодеятельности:

```scala
def obj: Parser[JSON] = (
  token("{") *>
  keyval.sep(token(",")).map(kvs =>
    JObject(kvs.toMap)
  ) <*
  token("}")
).scope("object")

def keyval: Parser[(String, JSON)] = escapedQuoted ** (token(":") *> value)
```
