## 209 Ответы на упражнения 8.6

Если размер домена меньше указанного числа тестовых случаев, то мы можем железно  
доказать наше свойство — как кота в коробке полностью проверили, все варианты  
перебрали. Иначе остаётся только сказать: «Не нашли убойный тест среди тех,  
что прогнали, но кто ж его знает, что там в тени прячется».  

Поэтому слегка перелопатим модель результата property-теста (property-based test),  
чтоб эта нюансированная хуйня была выразительной:

```scala
enum Status:
case Proven, Unfalsified
enum Result:
case Falsified(failure: FailedCase)
case Passed(status: Status, testCases: TestCases)
```

Результаты свойств теперь либо опровергнуты наглухо, либо прошли после определённого  
числа тестов, плюс флаг — доказано ли свойство на все сто (как в той теореме,  
что сам доказывал на код-ревью) или просто не нашли контрпримера, чтоб не расслаблялись.  

Возвращаясь к `forAll`, надо стратегию для тестовых случаев выбрать, чтоб не как  
лох на первом собеседовании. Один вариант — сначала все сэмплы из исчерпывающего  
списка прогнать, до лимита тестов. Дошли до конца без `None` или фейла на оценке  
сэмпла — свойство доказано, домен весь выебан. А если `None` вылез — догенерим  
рандомные тесты через `sample`.  

Но вот засада, пацаны: если `TestCases` меньше домена, то каждый запуск — те же  
самые тесты, как заезженная пластинка в продакшене. Фиксим, ограничив исчерпывающий  
список третью от тестов, остальное — рандом, чтоб как в жизни, с сюрпризами.  

В этой имплементации `forAll` именно так и сделали — треть от exhaustive (исчерпывающего),  
остальное рандом, чтоб не скучно:

```scala
def forAll[A](a: Gen[A])(f: A => Boolean): Prop =
(max, n, rng) =>
def go(i: Int, j: Int, l: LazyList[Option[A]],
onEnd: Int => Result
): Result =
if i == j then Passed(Unfalsified, i)
else l match
case Some(h) #:: t =>
try
if f(h) then go(i+1, j, t, onEnd)
else Falsified(h.toString)
catch
case e: Exception => Falsified(buildMsg(h, e))
case None #:: _ => Passed(Unfalsified, i)
case _ => onEnd(i)
val numFromExhaustiveList = TestCases.fromInt(n.toInt / 3)
go(0, numFromExhaustiveList, a.exhaustive, i => Passed(Proven, i)) match
case Passed(Unfalsified, _) =>
val rands = randomLazyList(a)(rng).map(Some(_))
go(numFromExhaustiveList, n, rands, i => Passed(Unfalsified, i))
case s => s // If proven or failed, stop immediately
```
