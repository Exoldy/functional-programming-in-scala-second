## 93 4.6 Ответы на упражнения

```scala
def sequence[E, A](as: List[Either[E, A]]): Either[E, List[A]] =
  traverse(as)(x => x)

def traverse[E, A, B](as: List[A])(f: A => Either[E, B]): Either[E, List[B]] =
  as.foldRight[Either[E, List[B]]](Right(Nil))((a, b) => f(a).map2(b)(_ :: _))
```

Кроме сигнатур типов, единственное изменение — наш выбор начального аккумулятора: `Right(Nil)` вместо `Some(Nil)`. Пацаны, такая мелочь, а в продакшене спасает от тихого пиздеца с Nil в хвосте.
