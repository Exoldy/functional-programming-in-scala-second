## Functional Programming in Scala Second Edition

Michael Pilquist, Rúnar Bjarnason, Paul Chiusano

**Функциональный** код — это как танк: легко тестить, переиспользовать, параллелить, и он хуярит целые армии багов со стейтом, не моргнув глазом. С мощными FP-фичами, привычным синтаксисом и бесшовной (seamless) интеграцией с Java — Scala как раз то место, где новичок в FP не сломается на старте, а рванёт вперёд.

—Из Предисловия Дэниела Спивака, создателя Cats Effect

«Углуби своё понимание практического FP в Scala с этим ультимативным гайдом».

В **Functional Programming in Scala, Second Edition** ты изучишь функциональное программирование с нуля, по первым принципам. Практические упражнения и примеры помогут переключить мозг на функциональный лад. В этой переработанной редакции — тонны аннотаций к упражнениям, чтоб ты нырнул в FP по самые уши, плюс шаги по сборке своих функциональных либ в Scala. Как только лампочка FP загорится — кодить по-старому уже не захочешь, бро.

—Билл Веннерс, Artima

«Первое издание FPiS стало для меня поворотной точкой в кроличьей норе FP. Охуенно было доказывать, что один typeclass (typeclass)-интерфейс эквивалентен другому. Вторая редакция сохраняет уникальный вайб FPiS: ведёт через практические кодовые идиомы к математически строгому подходу в FP».

**Что внутри**

- Распознавай и пиши чисто функциональный код
- Работай с ошибками без этой хуйни под названием exceptions (exceptions)
- Копайся в стейте и concurrency (concurrency) без паники
- Взаимодействуй с функциональными структурами, которые задают общие behaviors (behaviors)

Для джавистов или скалистов. FP не обязателен на входе.

—Сергей Виницки, Workday
