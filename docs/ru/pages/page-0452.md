## 15.2 Простые трансформации стримов

Теперь мы можем выразить ядро трансформации для нашей задачки с подсчётом строк как `count` `andThen` `exists(_` `>` `40000)` — чисто, как в аптеке, и к пайплайну легко навешать фильтры с остальными трансформациями, будто Lego собираешь.

### 15.2.3 Обработка файлов

Изначальная хрень — проверить, больше ли в файле 40к строк — теперь решается на раз-два, без напряга. Но пока мы ковырялись только с чистыми стримами, как в вакууме. К счастью, файл можно впихнуть источником элементов в `Stream`, а вместо генерации `Stream` на выходе — собрать все потоки стримов в одно финальное значение, чтоб не распыляться.

**Листинг 15.3.** Использование `Stream` с файлами вместо `LazyList`

```scala
def fromIterator[O](itr: Iterator[O]): Stream[O] =
Pull.unfold(itr)(itr =>
if itr.hasNext then Right((itr.next(), itr))
else Left(itr)
).void.toStream
def processFile[A](
file: java.io.File,
p: Pipe[String, A],
)(using m: Monoid[A]): IO[A] = IO:
val source = scala.io.Source.fromFile(file)
try fromIterator(source.getLines).pipe(p).fold(m.empty)(m.combine)
finally source.close()
```

Функция `processFile` открывает файл и использует `fromIterator`, чтоб слепить `Stream[String]` из строк файла. Потом прогоняет по нему твой пайп, получает `Stream[A]` и редусирует этот `Stream[A]` в одно `A` — фолдом по выходам с помощью `Monoid[A]`. Всю эту оргию оборачивает в `IO`, чтоб `processFile` оставалась референциально прозрачной (referentially transparent), как слеза девственницы. Теперь изначальную задачу валим вот так:

![Страница 0452, изображение 1](./assets/page-0452/image-01.png)

```scala
def checkFileForGt40K(file: java.io.File): IO[Boolean] =
processFile(file, count andThen exists(_ > 40000))(using Monoid.booleanOr)
```

#### УПРАЖНЕНИЕ 15.8

Напиши программу, которая читает градусы Фаренгейта (Fahrenheit) как `Double` из файла — по одному значению на строку, прогоняет каждое через пайплайн, конвертит в градусы Цельсия (Celsius) и пишет результат в другой файл. Игнорируй пустые строки в инпуте, а также те, что стартуют с символа `#`. Можешь использовать функцию `toCelsius`:

```scala
def toCelsius(fahrenheit: Double): Double =
(5.0 / 9.0) * (fahrenheit - 32.0)
def convert(inputFile: String, outputFile: String): IO[Unit]
```
