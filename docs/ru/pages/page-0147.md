## Чисто функциональное состояние

### Что в этой главе

- Обсуждаем чисто функциональную генерацию рандома
- Работаем с stateful API
- Знакомимся с типом данных `State`

В этой главе разберёмся, как писать чисто функциональные проги, которые ковыряются в состоянии, — на примере простой домены *генерации случайных чисел*. Сам по себе рандом — не самый заебись кейс для этих техник, но его простота как раз то, что доктор прописал для первого раза, чтоб не утонуть в complexity с ходу. Более сочные примеры ждут в частях 3 и 4 книги, особенно в 4-й, где мы глубоко нырнём в состояние и эффекты по самые уши. А цель тут простая: дать тебе базовый паттерн, чтоб любой stateful API перевернуть с ног на голову в чистый FP. Когда сам начнёшь лепить свои функциональные API, нарвёшься на те же грабли, что мы тут раскидали, — поверь, я через это прошёл, знаю подвохи.
