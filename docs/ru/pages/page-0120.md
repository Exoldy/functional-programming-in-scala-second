## 91 Ответы на упражнения 4.6

Сначала берём среднее от исходных сэмплов — бац, и у нас в руках `Option[Double]`, как манна небесная для FP-шника. Далее вызываем на ней `flatMap`, впаривая анонимку типа `Double` `=>` `Option[Double]` — ну чисто лямбда-магия из арсенала, который все мы ковыряли на код-ревью до посинения. Внутри этой хрени маппим по оригинальным сэмплам, каждый элемент `x` крутим-вертим в `math.pow(x` `-` `m,` `2)`, а потом хватаем `mean` от этой подкрученной последовательности — как кофе после ночи кодинга. Заметьте, внутренняя `mean` возвращает `Option[Double]`, поэтому пришлось `flatMap` результат внешней `mean`, чтоб не утонуть в матрёшке Option'ов. А если б вместо неё `map` впихнули — привет, `Option[Option[Double]]`, и сиди разгребай, как в том меме с "now you have two problems".

#### ОТВЕТ 4.3

```scala
def map2[A, B, C](a: Option[A], b: Option[B])(f: (A, B) => C): Option[C] =
a.flatMap(aa => b.map(bb => f(aa, bb)))
```

Эта имплементация юзает `flatMap` на первом Option'е и `map` на втором — просто, как два пальца, без лишней хуйни. Если б `map` на обоих запустили, то нарвались бы на `Option[Option[C]]`, которую потом пришлось бы сплющивать через `getOrElse(None)`, а `map(g).getOrElse(None)` — это и есть суть `flatMap`, короче, классический FP-цикл "сделал сложнее, чтоб потом упростить". Альтернатива — паттерн-матчинг, без соплей:

```scala
def map2[A, B, C](a: Option[A], b: Option[B])(f: (A, B) => C): Option[C] =
(a, b) match
case (Some(aa), Some(bb)) => Some(f(aa, bb))
case _ => None
```

#### ОТВЕТ 4.4

```scala
def sequence[A](as: List[Option[A]]): Option[List[A]] =
as.foldRight[Option[List[A]]](Some(Nil))((a, acc) => map2(a, acc)(_ :: _))
```

Правый фолд по списку Option'ов, с `map2` в роли zero, а list cons — наша комбайн-машина для склейки. Тут стандартная либовая `List`, не та, что мы сами ковыряли раньше — имена другие, но суть та же: `::` вместо `Cons`, а `h` `::` `t` — это аналог `Cons(h,` `t)` из главы 3, только от Scala-отцов, без самопальных подвохов.

#### ОТВЕТ 4.5

```scala
def traverse[A, B](as: List[A])(f: A => Option[B]): Option[List[B]] =
as.foldRight(Some(Nil): Option[List[B]])((a, acc) =>
map2(f(a), acc)(_ :: _))
```

Для `traverse` та же тактика, что и для нашей первой версии `sequence`: правый фолд по элементам списка, с `map2` и `::` в роли комбайна
