> Часть 4: Эффекты и I/O / Глава 15: Обработка потоков и инкрементальный I/O / 15.2 Простые трансформации потоков / 15.2.1 Создание pull'ов

В главе 5 мы юзали ленивую оценку, чтоб строить ленивые списки, а тут полагаемся на лень `flatMap` — чтоб хвост не вычислять сразу, а отложить на потом, как истинный лентяй в продакшене. 

Давай обобщим этот рекурсивный паттерн в более высокоуровневый конструктор pull'а, чтоб не ебаться с копипастой:

```scala
def unfold[O, R](init: R)(f: R => Either[R, (O, R)]): Pull[O, R] =
f(init) match
case Left(r) => Result(r)
case Right((o, r2)) => Output(o) >> unfold(r2)(f)
```

`unfold` стартует с начального сида типа `R` и итеративной функцией, которую гоняем по кругу: она либо выдаёт финальный результат типа `R`, либо плётёт выход `O` плюс новый сид `R`. Это чисто та же хрень, что мы ковыряли в упражнении 5.11 для `LazyList` — помните, как там unfold разворачивал мир в бесконечность?

#### УПРАЖНЕНИЕ 15.1

Заточь `fromListViaUnfold` и `fromLazyListViaUnfold` как методы в компаньоне `Pull`, используя `unfold`:

```scala
def fromListViaUnfold[O](os: List[O]): Pull[O, Unit]
def fromLazyListViaUnfold[O](os: LazyList[O]): Pull[O, Unit]
```

Как мы делали для `LazyList`, можно слепить конструктор `continually`:

```scala
def continually[A](a: A): Pull[A, Nothing] =
Output(a) >> continually(a)
```

Тип возврата у `continually` — это вообще засада: результат `Nothing`, потому что pull бесконечный, как баг в legacy-коде, и никогда не завершится значением. 

Вытащим рекурсию в общий комбинатор, чтоб не дублировать:

```scala
enum Pull[+O, +R]:
def repeat: Pull[O, R] =
this >> repeat
```

![Страница 0445, изображение 2](./assets/page-0445/image-02.png)

```scala
object Pull:
def continually[A](a: A): Pull[O, Nothing] =
Output(a).repeat
```

#### УПРАЖНЕНИЕ 15.2

Определи `iterate` как метод в компаньоне `Pull`. Должен вернуть бесконечный поток, который сначала выдаст начальное значение, потом применит к нему функцию, потом к результату — и так по кругу, как кофе-машина в офисе:

```scala
def iterate[O](initial: O)(f: O => O): Pull[O, Nothing]
```
