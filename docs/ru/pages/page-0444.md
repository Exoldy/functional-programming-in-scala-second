## 415 15.2 Простые трансформации потоков

`Pull[O, R]` — это как конвейер-убийца: сыплет сколько влезет значений типа `O`, а потом бац — один финальный `R` и сдох. Его ковыряешь по шагам через `step`: либо сразу `R` выдал и похуй, либо `O` на пол и новый `Pull` на остаток стримa. С такой моделью лепишь любые ебанутые pull'ы из кучи источников, цепляя `Pull.Output` через `flatMap`, как Lego на стероидах. Когда пора запускать трансформацию стримa — хуярь `fold`, он степит до посинения, пока не сдохнет. Особо заебись `flatMap`: не превращает каждый `O` в новый pull и не конкатит их в бесконечный пиздец, а работает с финальным `R`. Из-за этого два pull'а склеиваешь через заимствованный `>>`, как два вагона поезда. Ща разберем. `Pull` ковариантен и в выводе, и в результате — магия, братцы. Благодаря этому `Result[+R]` — это просто `Pull[Nothing, R]`, а `Output[+O]` — `Pull[O, Unit]`. Похоже на то, как `None` — `Option[Nothing]`, а `Nil` — `List[Nothing]`. API от этого inference'у сосет не так сильно. Глубже в типы инференса нырнем позже, не ссыте.

### 15.2.1 Создание pull'ов

Один из самых простых pull'ов — тот, что нихуя не выдает и с unit'ом до свидания:

```scala
val done: Pull[Nothing, Unit] = Result(())
```

Можем лепить pull'ы с одиночными значениями и их конкатить, как матрешку:

```scala
scala> val p = Pull.Output(1) >> Pull.Output(2)
val p: Pull[Int, Unit] = FlatMap(Output(1),...)
scala> val q = p.toList
val q: List[Int] = List(1, 2)
```

Операция `p1 >> p2` — это сахар для `p1.flatMap(_ => p2)`. Рекурсией с `done` и `>>` строим pull'ы из списков и ленивых списков, стек не жрет:

```scala
def fromList[O](os: List[O]): Pull[O, Unit] =
os match
case Nil => done
case hd :: tl => Output(hd) >> fromList(tl)
def fromLazyList[O](os: LazyList[O]): Pull[O, Unit] =
os match
case LazyList() => done
case hd #:: tl => Output(hd) >> fromLazyList(tl)
```

В обоих `fromList` и `fromLazyList` рекурсим: каждый элемент в `Output`-pull, потом `>>` и лениво дальше. Стек-safe чисто из-за ленивого аргумента в `>>` — классика, как в старые времена с ленивыми
