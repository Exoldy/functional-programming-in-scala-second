#### ОТВЕТ 15.8

Сначала давай сварганим pipe ``Pipe[String,`` ``Double]``, который хватит строку с вводом и перехуярит её в Цельсии — как тот алхимик, что из Fahrenheit'овской жижицы выжимает нормальную температуру. Определим его как цепочку из микро-pipe'ов, каждый с одной хуёвой задачей на уме, чтоб не плодить монстров Франкенштейна, а потом склеим в один эпичный поток, как Lego из FP-кирпичиков:

```scala
def trimmed: Pipe[String, String] =
src => src.map(_.trim)
def nonComment: Pipe[String, String] =
src => src.filter(_.charAt(0) != '#')
def asDouble: Pipe[String, Double] =
src => src.flatMap: s =>
s.toDoubleOption match
case Some(d) => Stream(d)
case None => Stream()
def convertToCelsius: Pipe[Double, Double] =
src => src.map(toCelsius)
val conversion: Pipe[String, Double] =
trimmed andThen
nonEmpty andThen
nonComment andThen
asDouble andThen
convertToCelsius
```

Дальше берём этот pipe и прогоняем через него строки из входного файла, конвертим на лету и сливаем каждую в выходной — чистый конвейер, блядь, без лишнего копипаста. Для этого нужен драйвер в типе ``IO``: открываем файлы на чтение/письмо, цепляем конвертацию к строкам ввода и шлём трансформированные прямиком в аутфайл, чтоб не париться с промежуточными буферами:
