## 17 2.1 Знакомство с языком Scala

Мы объявляем объект по имени `MyProgram`. Это просто чтобы дать нашему коду тёплый уголок и имя на двери, чтоб потом не рыскать по файлу, как по помойке. Внутрь запихиваем код отступами для каждого *члена* (member).[^1] Объекты и классы разберём по косточкам чуть позже. А пока ковыряемся в этом конкретном объекте, как в старом жёстком диске.

![Страница 0046, изображение 1](./assets/page-0046/image-01.png)

Ключевое слово `object` создаёт новый *синглтон-тип* (singleton type), который как `class`, только с одним-единственным экземпляром, как тот последний кусок пиццы на код-ревью. Если ты из Java-мира, то объявление `object` в Scala — это как слепить анонимный класс и инстанцировать его на лету. В Scala нет аналога Java-шному `static`, и объект часто юзают там, где в Java ты бы накидал статических *членов* (members) в класс — типа, "зачем плодить сущности, если можно одним выстрелом двух зайцев".

Объект `MyProgram` имеет три *метода* (methods), помеченных ключевым словом `def`: `abs`, `formatAbs` и `printAbs`. Под *методом* (method) будем понимать функцию, объявленную внутри объекта или класса с помощью `def` — классика жанра. Теперь пройдёмся по методам `MyProgram` по одному, как по баглисту. Метод `abs` — чистая функция, которая жрёт инт и выдаёт его абсолютное значение, без всякой мутации, как нормальный FP-пацан:

```scala
def abs(n: Int): Int =
if n < 0 then -n
else n
```

За `def` идёт имя метода, потом параметры в скобках. Тут `abs` принимает один аргумент: `n` типа `Int`. После закрывающей скобки — опциональная аннотация типа (`: Int`), мол, результат будет `Int` (двоеточие читается как "имеет тип"). Тело метода — после одинарного равно (`=`). Левую сторону от равно (сигнатуру) отличаем от правой (определения), чтоб не путаться, как в дебегах. Нет явного `return` — метод возвращает то, что вылилось из правой стороны. Все выражения в Scala дают результат, даже `if`. Тут правая сторона — if-ик, который выдаст либо `-n`, либо `n`, в зависимости от того, `n < 0` ли.

Метод `formatAbs` — ещё одна чистая функция, без подвохов:

![Страница 0046, изображение 2](./assets/page-0046/image-02.png)

```scala
private def formatAbs(x: Int) =
val msg = "The absolute value of %d is %d."
msg.format(x, abs(x))
```

> `format` — это стандартный метод из либы, висящий на строках.

Тут мы вызываем метод `format` на объекте `msg`, впихивая значение `x` и результат `abs` от `x`. В итоге получается свежая строка, где `%d` в `msg` заменены на результаты вычислений `x` и `abs(x)` соответственно — магия интерполяции, братан.

[^1]: Объект `MyProgram` тут опционален, потому что Scala позволяет *определения на верхнем уровне* (top-level definitions). Могли бы выкинуть `MyProgram` нафиг, но мы его юзаем, чтоб сгруппировать родственные *члены* (members) в кучку, как нормальные люди.
