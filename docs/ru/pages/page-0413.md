А потом лепим `given`-инстансы этого трейта под каждую монаду, которую хотим тащить в прод:

```scala
given parConsole: Console[Par] with
def readLn =
Par.lazyUnit(Try(readLine()).toOption)
def printLn(line: String) =
Par.lazyUnit(println(line))
given thunkConsole: Console[Function0] with
def readLn =
() => Try(readLine()).toOption
def printLn(line: String) =
() => println(line)
```

Эти инстансы — как те интерпретаторы, что мы ковыряли для алгебр ``Free``, но с подвохом: мы не строим сначала программу ``Free``, а потом её разгоняем интерпретатором, как в старые времена с Free-апокалипсисом. Нет, здесь сразу через `given`-инстансы целевую монаду материализуем — чистая алхимия Scala 3. Когда лепили программы ``Free``, дататип ``Free`` давал нам цепочку операций, как конвейер на заводе. А кто здесь обеспечивает эту последовательность в capability-подходе? Целевая монада сама, блядь, по умолчанию! Это вылазит, как только capability-трейт в деле берёшь — магия implicits на стероидах.

Давай зафигачим ту же программу приветствия, что раньше ковыряли, — только теперь с capability-трейтом ``Console[F]``:

```scala
def greet[F[_]](using c: Console[F], m: Monad[F]): F[Unit] =
for
_ <- c.printLn("What's your name?")
n <- c.readLn
_ <- n match
case Some(n) => c.printLn(s"Hello, $n!")
case None => c.printLn(s"Fine, be that way.")
yield ()
```

Функция ``greet`` теперь дефайнится для абстрактной целевой монады ``F``, у которой есть и инстанс ``Console``, и ``Monad`` — полная комбо. В теле лепим консольные операции, колля ``c.printLn`` и ``c.readLn``, а секвенсим их через `given`-инстанс ``Monad[F]``. Без параметра контекста ``Monad[F]`` хуй бы мы смогли колить ``flatMap`` на результатах ``printLn`` и ``readLn`` — ни хуя не знаем про тип ``F``, как в тёмном лесу без карты. Говорим, что ``Console`` наделяет конструктор типа ``F`` суперсилой ковырять консоль, а ``Monad`` вдалбливает в ``F`` секвенсинг (то бишь ``flatMap``). Вместо того чтобы тягать free-алгебры как чемоданы без колёс, мы пассим абстрактные эффектные программы — лёгкие, как перо Феникса. Когда припрут — инстансим для конкретного эффекта (типа ``greet[Function0]``) и гоняем получившуюся монаду, без лишнего boilerplate'а.
