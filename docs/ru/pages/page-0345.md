# Страница 0345
[<- Страница 0344](./page-0344) | [Индекс страниц](./) | [Страница 0346 ->](./page-0346)

> Часть 3: Общие структуры в функциональном дизайне / Глава 12: Аппликативные и траверсибельные функторы / 12.2 Трейт Applicative

```scala
extension [A](fa: F[A])
def map3[B, C, D](
fb: F[B],
fc: F[C])(f: (A, B, C) => D): F[D]
def map4[B, C, D, E](
fb: F[B],
fc: F[C],
fd: F[D])(f: (A, B, C, D) => E): F[E]
```

Мы могли бы определить `map3` и `map4` как обычные методы, а не extension methods (типа `def` `map3[A,` `B,` `C,` `D](fa:` `F[A],` `fb:` `F[B],` `fc:` `F[C])(f:` `(A,` `B,` `C)` `=>` `D):` `F[D]`). Один из минусов — места вызовов раздуваются в хуйню многословную, как отчёт после менеджерского код-ревью. Например, если `map3` определён как обычный метод на трейте `Applicative`, то суммировать три `Option[Int]` — это `summon[Applicative[Option]].map3(oa,` `ob,` `oc)(_` `+` `_` `+` `_)` вместо чистого `oa.map3(ob,` `oc)(_` `+` `_` `+` `_)`, когда `map3` — extension method. В этом случае Scala шарит по extension methods `map3` для `Option[Int]` и находит его на `Applicative[Option]`. Даже упоминать `Applicative` не надо — чистый кайф, как implicit resolution в лучшие времена Cats.

Более того, теперь можем сделать `Monad[F]` подтипом `Applicative[F]`, подсунув дефолтную реализацию `map2` через `flatMap`. Это прям намёк от типа-системы: все монады — аппликативные функторы по жизни, и нехуй дублировать инстансы `Applicative` для всех наших типов, у которых монады уже висят.

Listing 12.2 Делаем `Monad` подтипом `Applicative`

```scala
trait Monad[F[_]] extends Applicative[F]:
extesnion [A](fa: F[A])
def flatMap[B](f: A => F[B]): F[B] = fa.map(f).join
```

> Минимальная реализация Monad должна реализовать unit и переопределить либо flatMap, либо join и map.

```scala
extension [A](ffa: F[F[A]])
def join: F[A] = ffa.flatMap(identity)
def compose[A, B, C](f: A => F[B], g: B => F[C]): A => F[C] =
a => f(a).flatMap(g)
extension [A](fa: F[A])
def map[B](f: A => B): F[B] =
fa.flatMap(a => unit(f(a)))
def map2[B,C](fb: F[B])(f: (A, B) => C): F[C] =
fa.flatMap(a => fb.map(b => f(a, b)))
```

Пока что мы просто переставили местами функции API и плясали под типовые подписи, как на корпоративе под диджея. Давай отойдём на шаг назад, пацаны, и разберёмся в разнице выразительности между `Monad` и `Applicative` — и что за этим всем кроется на самом деле.
