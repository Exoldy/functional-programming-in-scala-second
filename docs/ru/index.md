# Функциональное программирование в Scala, Второе издание

Этот сайт слеплен прямиком из оригинального PDF — разбит по одной Markdown-страничке на файл, чтоб не ебаться с жирным монолитом в продакшене.

- Исходный PDF: `Functional_Programming_in_Scala_Second_.pdf`
- Сгенерированные страницы: [`./pages/`](./pages/)
- Английская версия: [`../`](../)
- Перегенерить локально: `npm run convert`

Боковушка тянется из оглавления PDF, а на каждой странице болтается навигация prev/next — чтоб не блудить, как в лабиринте Минтавра на код-ревью.
