```scala
case FlatMap(Suspend(fa), f) => fa.flatMap(a => f(a).run)
case FlatMap(_, _) =>
sys.error("Impossible, since `step` eliminates these cases")
```

Note that we need to take a `Monad[F2]` now, instead of a `Monad[F]`. The other operations—`step`, `runFree`, and `translate`—can be reused with no changes in signature. There are a few places in their definitions in which Scala needs some help with type inference: via type ascription, explicitly passing type parameters, or using `covary`. See the answer key in the GitHub repository for details.
