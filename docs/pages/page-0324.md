## 295 11.4 Monad laws

This function has the right type to be passed as an argument to `compose`.11 The effect should be that anything composed with `unit` is that same thing. This usually takes the form of two laws, *right identity* and *left identity*:

![Page 0324 image 1](./assets/page-0324/image-01.png)

> Right identity law

```scala
compose(f, unit) == f
compose(unit, f) == f
```

> Left identity law

![Page 0324 image 2](./assets/page-0324/image-02.png)

We can also state these laws in terms of `flatMap`, but they’re less clear that way:

> Right identity law

```scala
x.flatMap(unit) == x
unit(y).flatMap(f) == f(y)
```
