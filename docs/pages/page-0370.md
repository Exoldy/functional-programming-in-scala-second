We can make it a bit clearer that `mapN` can be implemented in terms of `apply` and `map(N` `-` `1)`, though we can no longer use `f.curried` for arity 3 or greater, which makes the syntax a bit noisy:

```scala
extension [A](fa: F[A])
def map[B](f: A => B): F[B] =
apply(unit(f))(fa)
def map2[B, C](fb: F[B])(f: (A, B) => B): F[C] =
apply(fa.map(f.curried))(fb)
def map3[B, C, D](
fb: F[B],
fc: F[C]
)(f: (A, B, C) => D): F[D] =
apply(fa.map2(fb)((a, b) => f(a, b, _)))(fc)
def map4[B, C, D, E](
fb: F[B],
fc: F[C],
fd: F[D]
)(f: (A, B, C, D) => E): F[E] =

```
