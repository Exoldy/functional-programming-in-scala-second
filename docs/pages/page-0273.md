#### ANSWER 9.7

Both `product` and `map2` can be implemented with for-comprehensions, which bind the results of `p` and `p2` and then combine those results:

```scala
extension [A](p: Parser[A])
def product[B](p2: => B): Parser[(A, B)] =
for
a <- p
b <- p2
yield (a, b)
def map2[B, C](p2: => B)(f: (A, B) => C): Parser[C] =
for
a <- p
b <- p2
yield f(a, b)
```

#### ANSWER 9.8

We can implement `map` using `flatMap` and `succeed`:

```scala
extension [A](p: Parser[A])
def map[B](f: A => B): Parser[B] =
p.flatMap(a => succeed(f(a)))
```

#### ANSWER 9.9

A JSON document starts with either an array or an object. Let’s start by sketching that, inventing new parsers as necessary:

```scala
def document: Parser[JSON] = array | obj
def array: Parser[JSON] = ???

```

Let’s also discard any whitespace at the start of a document. We can do so by writing a parser for whitespace and combining it with `array` `|` `obj`:

```scala
def whitespace: Parser[String] = Parser.regex("\\s*".r)
def document: Parser[JSON] = whitespace.map2(array | obj)((_, doc) => doc)
```
